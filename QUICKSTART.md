# 🎯 Быстрый старт

## ✅ Запуск приложения

### Вариант 1: Готовый EXE (рекомендуется)
```
1. Перейдите в папку: dist/
2. Запустите: TerminalCommandsGuide.exe
3. Готово!
```

### Вариант 2: Из архива
```
1. Распакуйте: dist/TerminalCommandsGuide-Portable.zip
2. Запустите TerminalCommandsGuide.exe из любой папки
3. Готово!
```

### Вариант 3: Из исходников
```bash
cd terminal_commands_app
pip install -r requirements.txt
python main.py
```

## 🆘 Решение проблем

### Проблема: Русский текст отображается некорректно
**Решение:** Обновитесь до версии 2.1+ (исправление включено)

### Проблема: Приложение не запускается
**Решение:** 
1. Проверьте наличие прав на выполнение
2. Добавьте в исключения антивируса
3. Запустите от имени администратора

### Проблема: Команда не выполняется
**Решение:**
1. Проверьте права администратора
2. Некоторые команды требуют повышенных привилегий
3. Попробуйте выполнить вручную в терминале
